# Product Management Tools
![Project State](https://img.shields.io/badge/state-early--stage_•_actively--evolving-blueviolet)


A curated repository of frameworks, templates, playbooks, and insights for effective product management practice and continuous learning. This repo exists as a working library of the models and thoughts I use to drive clarity, sequencing, and strategy across product teams.

There is focus on both practicality and theory. Tools that help PMs make better decisions today; Abstract ideas to digest, ponder, and adapt. 

---

## Purpose

This repo serves two audiences:

- **My own practice** as a version controlled home for the tools I refine over time.
- **Product peers** whol may find reusable artefacts for discovery, prioritisation, and decision making.

Everything here is designed to be portable, teachable, and actionable. 

---

## Philosophy

These tools reflect a few core beliefs about effective product management:

- **Clarity beats complexity** - Simple tools used consistently outperform ornate frameworks used rarely.
- **Narrative first** - If you can't explain the problem cleanly, you can't solve it cleanly.
- **Judgement is the differentiator** - Frameworks *support* decisions; they don't replace them.
- **Evidence over opinion** - Evidence includes user behaviour, operational friction, and system constraints, as well as "data".
- **Teachability matters** - A good PM scales themself by making their thinking legible.
